Skill Tree: Visual Concepts

Which tech tree direction feels right for DevOps Learn?

A

Vertical Tech Tree (like Path of Exilion)

Scrollable vertical tree. Start at the bottom with Docker basics, branch upward into Compose, K8s, DevOps. Locked nodes are dimmed, completed ones glow. Clean lines connect prerequisites.

B

Horizontal Skill Tree (like Civics Tree)

Left-to-right progression. Docker on the left, branching right into specializations. Columns represent difficulty tiers. Easier to see parallel paths and side-branches.

C

Radial / Hub Tree

Center node = "DevOps Beginner". Branches radiate outward in 4 directions (Docker, Compose, K8s, DevOps). Each branch grows outward with more advanced skills. Feels like exploring a map.
