# ما هي الحاويات؟

الحاويات هي حزم خفيفة الوزن ومستقلة تتضمن كل ما يلزم لتشغيل برنامج — الكود، وقت التشغيل، أدوات النظام، المكتبات، والإعدادات. على عكس الآلات الافتراضية، تشترك الحاويات في نواة نظام التشغيل المضيف، مما يجعلها أكثر كفاءة.

## الحاويات مقابل الآلات الافتراضية

| الميزة | الحاويات | الآلات الافتراضية |
|--------|---------|-----------------|
| بدء التشغيل | ثوانٍ | دقائق |
| الحجم | ميغابايت | غيغابايت |
| العزل | مستوى العملية | مستوى الأجهزة |

## تشغيل حاويتك الأولى

```bash
docker run -d -p 8080:80 nginx
```
